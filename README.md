# base-space

> 基于 Vue2 的 BaseSpace 组件

## 规划

- [ ] 组件开发
- [ ] 示例文档
- [ ] 组件打包

## 安装

```bash
npm i base-space
```

## 使用

```js
import BaseSpace from 'base-space'

export default {
  components: {
    BaseSpace
  }
}
```

## 示例

```vue
<template>
  <div>
    <base-space />
  </div>
</template>
```
